About the Project

This project implements a hybrid machine learning framework combining Random Forest and Support Vector Machines (SVM) for advanced DDoS detection in Software-Defined Networks (SDN).

We designed this system to accurately detect DDoS attacks by analyzing network traffic patterns. The hybrid approach leverages the strengths of both Random Forest and SVM for improved accuracy and reduced false positives. The system can quickly identify abnormal traffic and help mitigate attacks in real-time.

The detection process is powered by a dual-plane architecture where both control plane and data plane traffic are analyzed separately to identify different types of DDoS threats. This enables the system to detect volumetric attacks, flooding, and anomalies within network operations.

Key Features:

  • ✅ Hybrid Machine Learning Framework combining SVM and Random Forest for robust detection
  • ✅ Real-Time DDoS Detection - Identifies and mitigates attacks as they happen
  • ✅ Dual-Plane Traffic Analysis - Analyzes both control and data plane traffic for a comprehensive view
  • ✅ User-Friendly Interface - Easy-to-navigate web interface for dataset upload, model training, and predictions
  • ✅ Scalability - Designed to handle large-scale network infrastructures efficiently

This system uses SelectKBest for feature selection, StandardScaler for data normalization, and LabelEncoder for encoding categorical variables to prepare the data for machine learning. The model is trained on historical network traffic data, and the system predicts whether incoming traffic is normal or an anomaly.

With a web-based interface built using Flask and styled with Bootstrap, users can easily upload datasets, view model accuracy, and make predictions on live traffic data. The backend is powered by MySQL, ensuring fast and secure data storage.
